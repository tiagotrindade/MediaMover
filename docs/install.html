Choose How to Open FolioSort

FolioSort is not notarized by Apple, so macOS will block it on first launch.
Pick the method that matches your macOS version.

Recommended

Fastest Terminal — One Command

Works on every macOS version. Takes 5 seconds.

macOS 14 Sonoma & 15+ Sequoia
1 Drag FolioSort.app to your Applications folder
2 Open Terminal (Spotlight → type "Terminal")
3 Paste this command and press Enter:
xattr -cr /Applications/FolioSort.app
4 Done — double-click FolioSort to open. Works forever after this.

Alternative Methods

No Terminal System Settings → Open Anyway

For macOS 15 Sequoia and later. No Terminal needed.

macOS 15+ Sequoia
1 Drag FolioSort.app to your Applications folder
2 Double-click FolioSort — macOS will block it. Click "Done"
3 Open System Settings → Privacy & Security
4 Scroll down to find "FolioSort was blocked" and click "Open Anyway"
5 Confirm with your password or Touch ID. FolioSort will launch.
Simple Right-Click → Open

The classic method. Only works on macOS 14 Sonoma.

macOS 14 Sonoma only
1 Drag FolioSort.app to your Applications folder
2 Right-click (or Control-click) on FolioSort in Applications
3 Select "Open" from the context menu
4 Click "Open" in the security dialog that appears
5 Done — FolioSort will launch. Future opens work with a double-click.
⚠ Does NOT work on macOS 15 Sequoia. Apple removed this bypass in Sequoia. Use the Terminal or System Settings method above instead.
Troubleshooting — still can't open?

If none of the methods above work, try these steps:

1. Check that the app is in Applications:
Make sure you dragged FolioSort.app out of the DMG and into /Applications (or another folder). You cannot run apps directly from a DMG.

2. Re-run the xattr command with sudo:
sudo xattr -cr /Applications/FolioSort.app
Enter your Mac password when prompted.

3. Check System Settings:
Go to System Settings → Privacy & Security → App Management and make sure FolioSort isn't listed as blocked.

4. Re-download the DMG:
Occasionally the download can be corrupted. Try downloading the latest version from the releases page.
